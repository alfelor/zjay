Introduction to CSS3

Cascading Style Sheets (CSS) create a consistent look across webpages and separate structure from presentation.

  • CSS Level 1 published in 1996
  • CSS Level 2 published in 1998
  • CSS3 is modular with over 50 modules
  • CSS3 is backward compatible with CSS 2.1

CSS Solved a Big Problem

HTML was never intended to contain formatting tags. The addition of tags like <font> created challenges:

  • Maintaining large websites became difficult
  • Style information was duplicated on every page
  • W3C created CSS to solve these problems

CSS Syntax and Selectors

A CSS rule-set consists of a selector and a declaration block:

selector {
  property: value;
}

Element Selector

p {
  text-align: center;
  color: red;
}

ID Selector

#para1 {
  text-align: center;
  color: red;
}

Class Selector

.center {
  text-align: center;
  color: red;
}

Grouping Selectors

h1, h2, p {
  text-align: center;
  color: red;
}

Three Ways to Insert CSS

1. External Style Sheet

<head>
  <link rel="stylesheet" href="styles.css">
</head>

2. Internal Style Sheet

<head>
  <style>
    body { background-color: linen; }
  </style>
</head>

3. Inline Styles

<h1 style="color:blue;margin-left:30px;">
  This is a heading
</h1>

Cascading Order

  1. Inline styles
  2. Internal styles
  3. External styles
  4. Browser settings
  5. Browser defaults
