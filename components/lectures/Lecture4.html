HTML Basics: Lists, Links & Images

HTML Lists

Ordered Lists (Numbered)

Use <ol> for numbered lists:

  1. First item
  2. Second item
  3. Third item
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Unordered Lists (Bulleted)

Use <ul> for bulleted lists:

  • Apples
  • Bananas
  • Oranges
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>

Description Lists

Use <dl> for terms and definitions:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Navigation Menu

  • Home
  • About
  • Services
  • Contact

HTML Links

Basic Link

Create links with the <a> tag:

Visit Example.com

<a href="https://www.example.com">Visit Example.com</a>

Link Types

Type Description Example
Absolute Links to another website <a href="http://google.com">Google</a>
Relative Links within your site <a href="about.html">About</a>
Internal Jumps to section on same page <a href="#section1">Section 1</a>

Image as Link

<a href="#">
  <img src="image.jpg" alt="Description">
</a>

HTML Images

Basic Image

Add images with the <img> tag:

<img src="image.jpg" alt="Description of image">

Note: Always include the alt attribute for accessibility.

Image Formats

Format Best For
JPEG Photographs
PNG Graphics with transparency
GIF Simple animations
WebP Modern web (small file size)

Image Floating

Background Image

body {
  background-image: url('background.jpg');
}

Summary

  • Lists: Use <ol>, <ul>, and <dl> to organize content
  • Links: Create navigation with <a> tags and href attribute
  • Images: Add visuals with <img> tag and src attribute
  • Always include alt text for images
  • Use CSS to style lists and position images
