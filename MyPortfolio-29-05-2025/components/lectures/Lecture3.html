CIP1101 Lecture 3: HTML Tables

Topic: Lists and Tables

Defining an HTML Table

HTML tables display information in rows and columns. Basic elements:

  • <table> - Table container
  • <tr> - Table row
  • <th> - Table header (bold and centered)
  • <td> - Table data cell

Basic Table Example

Firstname Lastname Age
Nicole Pascual 18
Allen Ramirez 18
<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Nicole</td>
    <td>Pascual</td>
    <td>18</td>
  </tr>
</table>

Table Styling

Adding Borders

By default, tables have no borders. Add with CSS:

table, th, td {
  border: 1px solid black;
}

Collapsed Borders

Combine borders into a single border:

table {
  border-collapse: collapse;
}

Cell Padding

Space between content and borders:

th, td {
  padding: 10px;
}

Border Spacing

Space between cells:

table {
  border-spacing: 5px;
}
Note: Border spacing doesn't work with collapsed borders.

Advanced Table Features

Column Spanning (colspan)

Make a cell span multiple columns:

Company Name Telephone
The Travel Club (02) 332-7183 (02) 815-7418
<table>
  <tr>
    <th>Company Name</th>
    <th colspan="2">Telephone</th>
  </tr>
  <tr>
    <td>The Travel Club</td>
    <td>(02) 332-7183</td>
    <td>(02) 815-7418</td>
  </tr>
</table>

Row Spanning (rowspan)

Make a cell span multiple rows:

Company Name: The Travel Club
Telephone: (02) 332-7183
(02) 815-7418
<table>
  <tr>
    <th>Company Name:</th>
    <td>The Travel Club</td>
  </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
    <td>(02) 332-7183</td>
  </tr>
  <tr>
    <td>(02) 815-7418</td>
  </tr>
</table>
