CIP1101 Lecture 7: Bootstrap

Topic: Responsive Web Design Framework

What is Bootstrap?

Bootstrap is the most popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first websites.

Why Use Bootstrap?

  • Mobile-first approach
  • Excellent browser support
  • Easy to get started with HTML/CSS knowledge
  • Responsive design that works on all devices
  • Built-in components and utilities
  • Open source and free to use

Getting Started with Bootstrap

CDN Setup

Add these links to your HTML head section:

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Basic Template

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Bootstrap Page</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <!-- Bootstrap CSS CDN -->
</head>
<body>
  <div class="container">
    <h1>My Bootstrap Page</h1>
    <p>Resize to see responsiveness!</p>
  </div>
  <!-- Bootstrap JS CDN -->
</body>
</html>

Bootstrap Containers

Fixed Container

<div class="container">
  <!-- Content here -->
</div>

Responsive fixed-width container that changes at breakpoints.

Fluid Container

<div class="container-fluid">
  <!-- Content here -->
</div>

Full-width container spanning the entire viewport.

Container Breakpoints

Screen Size Container Width
<576px 100%
≥576px 540px
≥768px 720px
≥992px 960px
≥1200px 1140px

Bootstrap Grid System

The grid system uses a series of containers, rows, and columns to layout content.

Basic Grid Structure

<div class="container">
  <div class="row">
    <div class="col">Column 1</div>
    <div class="col">Column 2</div>
    <div class="col">Column 3</div>
  </div>
</div>

Responsive Grid Classes

  • .col- (extra small devices)
  • .col-sm- (small devices ≥576px)
  • .col-md- (medium devices ≥768px)
  • .col-lg- (large devices ≥992px)
  • .col-xl- (extra large ≥1200px)
